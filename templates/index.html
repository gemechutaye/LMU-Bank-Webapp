About LMU Bank

LMU Bank is an educational platform developed by Gemechu Taye for the "A Secure Webapp Project" (HOMEWORK #5).

This application demonstrates key web security principles:

  • Password hashing with pbkdf2
  • JWT authentication in HTTP-only cookies
  • Protection against CSRF, XSS, and SQL Injection
  • User enumeration defense

Manage your traditional and cryptocurrency banking accounts securely in this protected environment.
